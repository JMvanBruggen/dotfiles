# dotfiles
custom linux config files - managed via gnu stow

### Installation Linux
```sh
sudo apt install stow
git clone https://github.com/reinzor/dotfiles.git ~/.dotfiles
cd ~/.dotfiles
./setup.sh
```

### Installation OSX
```sh
brew install stow
git clone https://github.com/reinzor/dotfiles.git ~/.dotfiles
cd ~/.dotfiles
./setup.sh
```
