# dotfiles
custom linux config files - managed via gnu stow

### Installation Linux
```sh
sudo apt-get -y install stow curl && \
git clone https://github.com/reinzor/dotfiles.git ~/.dotfiles && \
cd ~/.dotfiles && \
./install.sh
```

### Installation OSX
```sh
brew install stow
git clone https://github.com/reinzor/dotfiles.git ~/.dotfiles
cd ~/.dotfiles
./install.sh
```
